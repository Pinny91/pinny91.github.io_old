The first game we created was a simple chess game for the mobile (with no AI). So it's possible to play with a friend on your smartphone.
This game was created because Tom, the 3D modeler, created a chessboard with the course he was following and I was kinda tired to follow courses and wanted to do something on my own.
Now the game is partially finnished but still some bugs to fix, for example on my phone it works fine but friends whoem installed it can't open it.
